Printer-Friendly:
Using the Stack Exchange Network API produces an essential printable view of a given question id with:
* All the answers
* All the comments

The easiest way to use this service is with:
* StackPrinter GreaseMonkey script
* Bookmarklet

Favorites:
Supported services:
1. The Stack Exchange Network
Retrieves and lets you print all the favorites users' questions

2. Delicious
Retrieves the latest 100 questions of a given Delicious username with stackoverflow tag (more info).
To add and tag your questions to Delicious use:
* Delicious GreaseMonkey script

TopVoted:
Retrieves and lets you print the top voted questions by tags
