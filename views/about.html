Printer-Friendly:
Produces an essential printable view of a given question id with:
* All the answers
* All the comments

The easiest way to use this service is with:
* StackPrinter GM script
* Bookmarklet

Favorites:
Supported services:
1. The Stack Overflow Trilogy
Retrieves and prints all the favorites users's questions

2. Delicious
Retrieves the latest 100 Trilogy questions of a given Delicious username with stackoverflow tag (more info).
To add and tag your questions to Delicious use:
* Delicious GM script

TopVoted:
Retrieves and prints the top voted questions by tags
